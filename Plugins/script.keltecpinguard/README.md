# Keltec Pin Guard

Keltec Pin Guard is an addon developed by **Keltec Media Play** for Kodi 19+ (Matrix, Nexus, Omega).

It allows you to set a PIN code that is required when navigating into protected TV Shows, Movies, Plugins or Repositories. There is also an option to force the user to enter a PIN on system startup, and restrict the amount of viewing time allowed per day.

Press **Back** to cancel the PIN entry and return to the previous screen — protected content remains locked.

## Features
- Block access to specific plugins, TV shows, movies, repositories and file sources
- PIN caching (optional) so you don't have to re-enter within a set time
- Multiple security levels
- User profiles with individual PINs and viewing time limits
- Time-based restrictions (PIN only active during certain hours)
